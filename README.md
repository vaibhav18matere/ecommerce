# Ecommerce App Using :

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Drizzle](https://orm.drizzle.team)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)

### Schema for

| Type          | Field          |
| ------------- | -------------- |
| 1. Users :    | userId         |
|               | email          |
|               | password       |
| 2. Products : | productId      |
|               | name           |
|               | description    |
|               | imageUrl       |
|               | quantity       |
|               | slug           |
| 3. Orders :   | userId         |
|               | address        |
|               | payment status |
|               | order status   |
